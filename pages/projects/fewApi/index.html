API к нескольким проектам

Этап: В процессе

О проекте:

Первый проект связан с написанием API к веб-сайту/мобильному приложению. Суть проекта заключалась в предоставлении возможности заказа услуг по уборке помещений. Для клиентов и работников было реализовано два API.

Второй проект связан с написанием API к мобильному приложению. Приложение упращает работу hr-менеджеров, так как позволяет делать удобные записи по назначенным интервью с тем или иным кандидатом, а также создавать этих кандидатов непосредственно из большого спектра файлов (поддерживаемые расширения резюме: rtf, docx, все расширения изображений, pdf).
